What's inside

Everything you need

Built by combining the best ideas from Flo, Clue, MyCalendar and Glow — with privacy at the core.

🌸

Cycle Prediction

Weighted moving average engine that adapts to your personal pattern. Predicts period, fertile window, and ovulation with confidence scoring.

📅

Interactive Calendar

Month view with color-coded period days, predicted window, fertile phase, ovulation marker, and PMS/luteal phase indicators.

📝

Full Daily Log

Flow, mood, 16 symptoms, pain level, BBT, cervical mucus, sleep, water intake, weight, exercise, and free-text notes.

📊

Insights & Charts

Cycle length trends, symptom frequency bars, mood patterns, pain charts, and flow history — all powered by fl_chart.

🤖

AI Chat Companion

Phase-aware health education via Groq Llama 3. The API key lives only in the Supabase Edge Function — never in the app.

🐰

Virtual Pet

A companion that grows with your tracking streak. Earn XP for logging, unlock 9 achievements, level up your pet.

💊

Birth Control

Daily pill check-in with streak tracking. Supports 8 methods: pill, patch, ring, IUD, implant, injection, condom.

🤰

Pregnancy Mode

Week-by-week tracker, due date calculator, kick counter with session logging, and trimester highlights.

🔒

Privacy First

No account required. All data stored locally. Optional cloud sync via Supabase. PIN + biometric app lock. Privacy mode.

How it's built

Clean Architecture

Strict separation of concerns. Presentation never touches Supabase or Firebase directly.

🎨 Presentation

  • Flutter widgets only
  • Consumes Riverpod providers
  • No direct backend calls
  • GoRouter navigation

⚙️ Domain

  • Pure Dart — no Flutter
  • CyclePredictionService
  • Business models
  • Repository interfaces

💾 Data

  • SharedPreferences (local)
  • Supabase (cloud sync)
  • Offline-first writes
  • Background sync queue

🔧 Core

  • 8 pastel palettes
  • Material 3 theme
  • Auth providers
  • Notification service

Features

Everything you need,
nothing you don't.

Built with the best ideas from Flo, Clue, and MyCalendar — but open source and ad-free.

🌸
Cycle Prediction
Weighted moving average engine with confidence scoring. Gets smarter with every cycle you log.
📅
Smart Calendar
Period days, fertile window, ovulation estimate, and PMS phase — all visualized beautifully.
📝
Daily Log
Flow, mood, symptoms, BBT, sleep, water, weight, cervical data — everything in one place.
📊
Insights & Charts
Cycle length trends, symptom frequency, mood patterns, BBT charts, and more.
🤖
AI Companion
Groq Llama model via Supabase Edge Function. Your API key never touches the client.
🐰
Virtual Pet
Earn XP for logging, level up your pet, unlock achievements. Health tracking made fun.
💊
Birth Control
Daily pill check-in with streak tracking. Supports pill, patch, ring, IUD, implant, and more.
🤰
Pregnancy Mode
Week-by-week tracker, kick counter, due date calculator, and appointment manager.
💑
Partner Sharing
Share a read-only cycle summary with a trusted partner via invite code.

Privacy

Your data. Your rules.

We built privacy in from the start — not as an afterthought.

📱
Offline First
All data stored locally on your device. No account required. Works without internet.
🔒
App Lock
PIN and biometric authentication. Privacy mode hides content in the app switcher.
🛡️
Row Level Security
Every Supabase table has RLS enabled. Users can only access their own data.
🗑️
Full Data Control
Export all your data as JSON or delete everything permanently at any time.
🤖
Safe AI
Groq API key never in the client. All AI calls go through a server-side proxy.
🚫
No Ads. No Tracking.
Open source. No analytics. No ad SDKs. No selling your health data.

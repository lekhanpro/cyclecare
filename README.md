# 🌸 CycleCare

A beautiful, privacy-first Android application for menstrual cycle tracking with 70+ features.

![Version](https://img.shields.io/badge/version-1.0.0-pink)
![Android](https://img.shields.io/badge/Android-7.0%2B-green)
![Build](https://img.shields.io/badge/build-passing-brightgreen)

## ✨ Features

- 📅 Interactive calendar with cycle predictions
- 📝 Daily mood and symptom tracking
- 📊 Comprehensive insights and analytics
- 🔔 Smart reminders and notifications
- 📤 Data export (CSV & PDF)
- 🔒 PIN lock & biometric authentication
- 🎨 Beautiful Material Design 3 UI
- 🌙 Dark mode support
- 🤰 Pregnancy mode
- 💊 Birth control tracking
- 📈 Health metrics tracking
- 🔐 100% local storage - your data never leaves your device

## 🚀 Download

Download the latest APK from the [Releases](https://github.com/lekhanpro/cyclecare/releases) page or from [GitHub Actions](https://github.com/lekhanpro/cyclecare/actions) artifacts.

## 🏗️ Tech Stack

- **Language:** Kotlin
- **UI:** Jetpack Compose + Material Design 3
- **Architecture:** MVVM + Clean Architecture
- **Database:** Room
- **DI:** Hilt (Dagger)
- **Async:** Coroutines + Flow
- **Notifications:** WorkManager
